# Projeto-web
Projeto da disciplina Implementação de Projetos Web
## Objetivo

Desenvolver um sistema web básico com HTML, CSS e JavaScript, aplicando conceitos de modelagem, requisitos e boas práticas de desenvolvimento.

## Estrutura

- `index.html`: Página principal
- `style.css`: Estilos da página
- `script.js`: Funcionalidades interativas
